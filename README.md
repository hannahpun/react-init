## Base Structure
If the project want to use CSS Module like me, the only thing is just follow the named rule PageXX.module.scss

```
├── src
  ├── layout
  ├── containers
      ├── PageXX
          ├── PageXX.module.scss  
          ├── PageXX.js
  ├── components
      ├── CompoXX
          ├── CompoXX.module.scss
          ├── CompoXX.js
```
